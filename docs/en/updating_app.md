# Updating the App

Therapets has an integrated system to keep your application always on the latest version without relying on external stores like Google Play.

## Automatic Updates

1. When a new version is released (published on our GitHub), the app detects it automatically.
2. A pop-up icon will appear on the main screen.
3. Tap the icon: the app will download the update and launch the native Android installer.

![Update Notice](/assets/images/app_update_en.png)

## Release Channels

You can choose what type of updates to receive by going to **Settings > Advanced Settings**:
- **Main:** Stable and recommended versions (Default).
- **Dev (Nightly):** Daily builds; they include the latest features but might have bugs.
- **Unstable:** Experimental versions.
